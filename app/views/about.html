This site performs some pre-configured searches against the GitHub Developer API to try to compile a list of repositories that are involved in the 18F Agile Delivery Services Blanket Purchase Agreement (BPA) Request For Quotation (RFQ) process.

It was created by M. Adam Kendall (LinuxBozo) as an experiment with Google's Material Design, and specifically Angular Material within AngularJS.
